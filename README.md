# LMS Project

A full-stack Learning Management System (LMS) built using Next.js App Router, MongoDB, NextAuth.js, and Tailwind CSS. This project allows students to register, enroll in courses, view lessons, and track progress. Admins can manage users, create courses, and monitor platform activity.

---

## 🚀 Features

- Secure authentication using NextAuth.js (Credentials & JWT)
- Student and Admin dashboards
- Course listing, enrollment, and progress tracking
- Modular reusable components (Navbar, CourseCard, etc.)
- Fully responsive UI using Tailwind CSS
- MongoDB integration using Mongoose

---

## 🧩 Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Database:** MongoDB + Mongoose
- **Authentication:** NextAuth.js (JWT-based)
- **Language:** JavaScript (no TypeScript)
- **Hosting (later):** Vercel
